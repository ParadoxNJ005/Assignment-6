### 6. MEASURES OF CENTRAL TENDENCY
```
Averages & Typical Values
в”Ӯ
в”ңв”Җв”Җ MEAN (Arithmetic Average)
в”Ӯ   в”ңв”Җв”Җ Formula: Sum of values / Count
в”Ӯ   в”ңв”Җв”Җ Data types: Interval, Ratio only
в”Ӯ   в”ңв”Җв”Җ Pros: Uses all data points
в”Ӯ   в”ңв”Җв”Җ Cons: Sensitive to outliers
в”Ӯ   в””в”Җв”Җ Use when: Normal distribution, no outliers
в”Ӯ
в”ңв”Җв”Җ MEDIAN (Middle Value)
в”Ӯ   в”ңв”Җв”Җ Calculation:
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Odd set: position (n+1)/2
в”Ӯ   в”Ӯ   в””в”Җв”Җ Even set: average of n/2 and (n/2)+1
в”Ӯ   в”ңв”Җв”Җ Data types: Ordinal, Interval, Ratio
в”Ӯ   в”ңв”Җв”Җ Pros: Unaffected by extreme outliers
в”Ӯ   в”ңв”Җв”Җ Best for: SKEWED DISTRIBUTIONS
в”Ӯ   в””в”Җв”Җ Use when: Outliers present, skewed data
в”Ӯ
в””в”Җв”Җ MODE (Most Frequent)
    в”ңв”Җв”Җ Data types: Nominal, Ordinal, Interval, Ratio
    в”ңв”Җв”Җ Pros: No outlier sensitivity
    в”ңв”Җв”Җ Cons: May not be unique
    в””в”Җв”Җ Use when: Categorical data
```

### 9. SKEWNESS DETAILS
```
Asymmetry of Distribution

SYMMETRICAL (NORMAL)
в”ңв”Җв”Җ Shape: Bell curve
в”ңв”Җв”Җ Mean = Median = Mode
в”ңв”Җв”Җ Skewness = 0
в””в”Җв”Җ No tail direction

POSITIVELY SKEWED (RIGHT SKEWED)
в”ңв”Җв”Җ Shape: Long tail on RIGHT
в”ңв”Җв”Җ Scores: Clustered lower
в”ңв”Җв”Җ Order: Mode < Median < Mean
в”ңв”Җв”Җ Tail pulls mean right
в””в”Җв”Җ Common in: Income data

NEGATIVELY SKEWED (LEFT SKEWED)
в”ңв”Җв”Җ Shape: Long tail on LEFT
в”ңв”Җв”Җ Scores: Clustered higher
в”ңв”Җв”Җ Order: Mean < Median < Mode
в”ңв”Җв”Җ Tail pulls mean left
в””в”Җв”Җ Common in: Test scores (ceiling effect)

CALCULATION
в”ңв”Җв”Җ Pearson's Coefficient
в””в”Җв”Җ = 3 Г— (Mean вҲ’ Median) / Std Dev
```

### 10. KURTOSIS DETAILS
```
Peakedness / Tail Behavior

MESOKURTIC
в”ңв”Җв”Җ Kurtosis = 0
в”ңв”Җв”Җ Normal distribution-like
в””в”Җв”Җ Standard peakedness

LEPTOKURTIC
в”ңв”Җв”Җ Heavy tails
в”ңв”Җв”Җ OUTLIERS PRESENT
в”ңв”Җв”Җ More peaked than normal
в”ңв”Җв”Җ Sharp peak, fat tails
в””в”Җв”Җ Extreme values likely

PLATYKURTIC
в”ңв”Җв”Җ Light tails
в”ңв”Җв”Җ NO OUTLIERS
в”ңв”Җв”Җ Flatter than normal
в”ңв”Җв”Җ Broad peak, thin tails
в””в”Җв”Җ Extreme values unlikely
```

### Distribution Shape вҶ’ Appropriate Measure
```
Normal Distribution
в”ңв”Җв”Җ Use: Mean (most accurate)
в”ңв”Җв”Җ Mean = Median = Mode
в””в”Җв”Җ No skew

Skewed Distribution
в”ңв”Җв”Җ Use: Median (robust)
в”ңв”Җв”Җ Outliers present
в””в”Җв”Җ Mean distorted by tails

Categorical Data
в”ңв”Җв”Җ Use: Mode (only option)
в””в”Җв”Җ Non-numerical values
```

### Skewness Indicators
```
Relationship Check
в”Ӯ
в”ңв”Җв”Җ If Mean > Median вҶ’ RIGHT SKEWED (positive)
в”ңв”Җв”Җ If Mean вүҲ Median вҶ’ NORMAL (symmetric)
в””в”Җв”Җ If Mean < Median вҶ’ LEFT SKEWED (negative)
```

## SUMMARY TABLE: DESCRIPTIVE STATISTICS AT A GLANCE

| Measure | Type | What it Shows | When to Use | Pros | Cons |
|---------|------|---------------|------------|------|------|
| **Mean** | Central Tendency | Average value | Normal dist., no outliers | Uses all data | Sensitive to outliers |
| **Median** | Central Tendency | Middle value | Skewed data, outliers | Robust | Ignores magnitudes |
| **Mode** | Central Tendency | Most common | Categorical data | Simple | May not be unique |
| **Range** | Variability | Max-Min spread | Quick overview | Easy to calculate | Only 2 data points |
| **Std Dev** | Variability | Avg distance from mean | Compare datasets | Intuitive | Units-dependent |
| **Skewness** | Shape | Asymmetry | Understand distribution | Identifies bias | Requires both mean & median |
| **Kurtosis** | Shape | Tail behavior | Detect outliers | Practical | Complex interpretation |

## KEY TAKEAWAYS
1. **EDA is iterative** - No fixed formula, explore systematically via 8 steps
2. **Data type matters** - Determines which statistics are valid
3. **Central tendency alone is insufficient** - Must assess variability/spread too
4. **Median > Mean for skewed data** - More robust to outliers
5. **Skewness & Kurtosis diagnose distribution** - Identify outliers and asymmetry
6. **Always visualize** - Histograms & scatterplots reveal patterns missed by numbers
7. **Handle outliers & missing data** - Steps 7-8 are crucial for data quality
