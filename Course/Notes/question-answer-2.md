# Comprehensive Data Visualization MCQ Quiz - Part 2
*Topics Covered: Measurement Scales, Geometry of Marks & Channels, Feature Extraction, Advanced Chart Types, Data Types, and Gestalt Theory.*

**Q1. Which of the following measurement scales possesses a true zero point, allowing for statements like "A is twice as large as B"?**
- A) Nominal
- B) Ordinal
- C) Interval
- D) Ratio

**Answer:** D) Ratio

---

**Q2. [Multi-Correct] According to Gestalt theory, which of the following are recognized principles of visual perception?**
- A) Principle of Enclosure
- B) Principle of Confusion
- C) Principle of Closure
- D) Principle of Figure and Ground

**Answer:** A, C, D

---

**Q3. In Descriptive Statistics, if a dataset's distribution has a long tail extending horizontally to the right, it is described as having:**
- A) Negative Skewness
- B) Positive Skewness
- C) Zero Skewness
- D) Platykurtic distribution

**Answer:** B) Positive Skewness

---

**Q4. Which chart type is specifically designed to show the distribution of a continuous numerical variable by grouping data into contiguous "bins"?**
- A) Bar Chart
- B) Histogram
- C) Scatterplot
- D) Pie Chart

**Answer:** B) Histogram

---

**Q5. [Multi-Correct] Which of the following are considered advantages of using a Waffle Chart over a traditional Pie Chart?**
- A) It avoids the difficulty the human eye has in visually estimating angles and circular areas.
- B) It represents continuous time-series data better.
- C) It is easier to compare small percentage differences accurately using discrete grid squares.
- D) It can handle datasets with hundreds of categories easily.

**Answer:** A, C

---

**Q6. A visualization that uses horizontal bars to illustrate a project schedule, showing the start and finish dates of various operational elements, is known as a:**
- A) Bump Chart
- B) Slope Chart
- C) Gantt Chart
- D) Cycle Chart

**Answer:** C) Gantt Chart

---

**Q7. In the context of Big Data, emails, social media posts, and video files are primary examples of what type of data?**
- A) Structured Data
- B) Semi-structured Data
- C) Unstructured Data
- D) Relational Data

**Answer:** C) Unstructured Data

---

**Q8. Which Feature Engineering technique converts continuous numerical values into targeted, grouped categories (e.g., converting ages 0-99 into "Child", "Adult", "Senior" brackets)?**
- A) One-Hot Encoding
- B) PCA (Principal Component Analysis)
- C) Discretization (Binning)
- D) Imputation

**Answer:** C) Discretization (Binning)

---

**Q9. [Multi-Correct] When utilizing Principal Component Analysis (PCA) during Feature Engineering, what are the primary goals?**
- A) To handle missing values by replacing them with the median.
- B) To reduce the dimensionality (number of variables) of high-dimensional data.
- C) To extract new features (components) that summarize the variance in the original variables.
- D) To scale all data to a strict range between 0 and 1.

**Answer:** B, C

---

**Q10. In Tamara Munzner’s analysis framework, "Length", "Area", "Color Hue", and "Spatial Position" are all examples of:**
- A) Geometric Marks
- B) Visual Channels
- C) Abstract Tasks
- D) Interaction Idioms

**Answer:** B) Visual Channels

---

**Q11. According to Munzner's ranking of visual encoding, which channel is considered the most effective for encoding categorical (identity) data?**
- A) Spatial Position
- B) Area (Size)
- C) Color Hue
- D) Color Saturation

**Answer:** C) Color Hue

---

**Q12. What specific visual technique is used primarily to track changes in rank order of entities across different time periods?**
- A) Slope Chart
- B) Bump Chart
- C) Streamgraph
- D) Area Chart

**Answer:** B) Bump Chart

---

**Q13. [Multi-Correct] Which of the following statements apply to Exploratory Data Analysis (EDA)?**
- A) It heavily utilizes visualizations to uncover hidden patterns and relationships.
- B) It tests pre-defined hypotheses using strict statistical thresholds like p < 0.05.
- C) It checks assumptions necessary for later statistical modeling.
- D) It is an iterative process often performed *before* formal statistical modeling begins.

**Answer:** A, C, D

---

**Q14. In visual hierarchy, what typography elements can be adjusted to immediately draw a viewer's pre-attentive focus to specific data insights?**
- A) Size, Weight (Boldness), and Color
- B) Using unformatted, matching paragraphs of text
- C) Removing all contrast from the dashboard
- D) Forcing all text to be uppercase and the exact same size

**Answer:** A) Size, Weight (Boldness), and Color

---

**Q15. Which encoding technique is necessary when converting categorical variables with NO inherent order (like "Red", "Green", "Blue") into binary variables to avoid implying an artificial hierarchy to a machine learning model?**
- A) Label/Integer Encoding
- B) Absolute Maximum Scaling
- C) One-Hot Encoding
- D) Decimal Scaling

**Answer:** C) One-Hot Encoding

---

**Q16. [Multi-Correct] Which of the following data scale types are considered "Quantitative" (numerical) rather than "Qualitative" (categorical)?**
- A) Nominal
- B) Ordinal
- C) Interval
- D) Ratio

**Answer:** C, D

---

**Q17. A visualization that cleanly compares data across exactly two different time periods using a line connected between two points for each category (often highlighting a specific comparative increase or decrease) is a:**
- A) Slope Chart
- B) Line Chart
- C) Connected Scatterplot
- D) Waterfall Chart

**Answer:** A) Slope Chart

---

**Q18. In visual encoding terminology, what is defined as the basic geometric primitives (0D, 1D, 2D) applied to represent data items or links?**
- A) Channels
- B) Marks
- C) Glyphs
- D) Annotations

**Answer:** B) Marks

---

**Q19. Which scaling technique divides every observation by the maximum mathematical value of that respective variable, resulting in values strictly bound between -1 and 1?**
- A) Min-Max Scaling
- B) Standardization
- C) Absolute Maximum Scaling
- D) Robust Scaling

**Answer:** C) Absolute Maximum Scaling

---

**Q20. [Multi-Correct] What are the primary limitations or disadvantages frequently cited concerning standard Pie Charts?**
- A) They become very difficult to read when visualizing a large number of categories.
- B) The human eye struggles to accurately compare relative angles and circular areas compared to straight lengths.
- C) They absolutely cannot display proportional or percentage parts of a whole.
- D) They take up significantly more dashboard space while conveying less precise comparative information than a simple bar chart.

**Answer:** A, B, D
